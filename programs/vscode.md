# Reconfigure

## reconfigure

reconfigure the user setting, located at `~/.config/Code/User/settings.json`

* auto save

  ```bash
    "files.autoSave": "onFocusChange"
  ```

* font size
  
    ```bash
    "editor.fontSize": 16,
    "editor.fontLigatures": false
    ```
